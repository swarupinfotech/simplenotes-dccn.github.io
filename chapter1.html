What is network Hardware and Software?
Network Hardware are electronic devices or network equipment required for communication and interaction between devices in a computer network. Ex :- Router, Hubs, Switches, bridge etc, While, Network Software are programs designed for efficient network performance and monitoring. It helps us to define locations of data storage and allow user to access that data. Hardware required + Software Required = Computer Networking Ex:- NIC Card(Network interface controller) :- It is a computer hardware component that connects a computer to connects a computer to computer network. Network driver is a software program that connect a computer network.

LINE CONFIGURATION :-
It refers to the way to connect physically two or more computer network.

Two types of physical connection:-
  • Point-to-point
  • Multipoint
Point-to-point
A point to point connection provides dedicated link between those two devices. Most point to point connection use, an actual length of wire or cable to connect the two ends, but other options such as microwave or satelite links are also possible.
Multipoint
multidrop configuration. In this connection, two or more device share links. more than the two device share the link that is capacity of the channel shared now. With shared capacity there can be two possibilities in a multipoint line configuration. spatial sharing:- if several devices can share the link simultaneously. it's called spatially shared line. configuration. Temporal (Time) sharing:- if users must link take turns using the link then it's called temporally shared or Time shared line configuration.

LIST OF DISADVANTAGES OF A COMPUTER NETWORKING:

1. It lacks independence. Computer networking involves a process that is operated using computer, so people will be relying more of a computer work, instead of exerting an effort for their tasks at hand. Aside from this, they will be dependent on main file server, which means that, if it breaks down , the system would become useless, making users idle.
2. It poses the security difficulties. Because there would be huge number of people who would be using computer network to get and share some of their files and resources, a certain user's security would be always at risk.

3. It allows for more presence of computer viruses and malware. There would be instance that stored files get corrupted due to computer viruses. Thus -network administrator should conduct regular check-ups on the and the stored files at same time.
4. It requires an efficient network administrator. For computer network to work efficiently and optimally, it requires high technical skills and know how of its operations and administrations. A person just having basic skill cannot do this job.
5. It lacks robustness. As previous stated, if a computer network's main server breaks down, the entire system would become useless. To deal with these problems, huge networks should have a powerful computer to serve as file server to make setting up and maintaining the network easier.
6. It requires an expensive set-up. Through the computer networks are said to be an inexpensive system when it I already running, its initial set up cost can still be high depending on the number of computers to be connected, Expensive devices, such as routers, switches, hubs etc., can add up to the cost,. Aside from these, it would also need network interface cards (NICs) for workstation in case they are not built in.
